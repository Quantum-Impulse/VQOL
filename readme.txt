The software in this folder was designed by Maria Maynard and Parth Shroff.

The main file is vqol.py in the MainFiles folder

To run, open up a command prompt on a computer running Python 3, change your directory to MainFiles, and enter

>> python vqol.py

To create an experiment, place the text file of the experiment you want to run in the Experiments folder.  Several example experiments are provided.

*** DO NOT MODIFY ANY OF THE FILES IN THE OTHER DIRECTORIES! ***
